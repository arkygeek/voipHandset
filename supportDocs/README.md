# HP Internet Handset — archived support documents

The PDFs in this directory are HP product documentation for the
HP Internet Handset (G9, USB `03f0:1c07`, HP part number GM320AA),
mirrored from HP's now-defunct support site for reference.

| File | Description |
|---|---|
| `HPInternetHandsetUsersGuide.pdf` | The 28-page user guide |
| `c01277321.pdf` | The product data sheet (1 page; the source of the "1 W >70 dBA SPL" ringer spec referenced throughout this project) |
| `c01277229.pdf` | "Important Information" — 4 pages of installation notes |
| `HPInternetHandsetPromo.pdf` | HP marketing material |
